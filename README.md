# media-support-thisismyurl
Media Hub for the thisismyurl.com Shared Code Suite - parent for image/video/audio/document processing plugins with shared media optimization and transcoding logic

## Modules

This hub provides the foundation for media processing and includes the following spoke modules:

### Documents Support (`/documents`)
Comprehensive document processing and management module for PDFs, DOCX, PPT, and other document formats. Features include:
- Document preview and inline viewing
- Full-text search within documents
- Version control and collections
- Document analytics and tracking
- Advanced features like format conversion and AI summaries

See [documents/README.md](documents/README.md) for complete feature list and documentation.
